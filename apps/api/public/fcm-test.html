SoLi FCM Push Test Page

Development tool — generates a browser FCM token and lets you send test push notifications.
Open this page from the NestJS server only: http://localhost:3000/fcm-test.html — not Live Server, not file://.
Requires a running backend at http://localhost:3000 with Firebase credentials configured.

Step 1 — Get FCM Token checking SW…

Firebase Messaging requires a registered service worker and notification permission. Click the button below to request permission and retrieve your browser FCM token.

Step 2 — Send Test Push via Backend API

Paste the token above into the field below (or it auto-fills), set a title and body, then click Send. The backend calls Firebase Admin SDK → FCM → your browser.
